# TestAssignmentSelectel
This is my test assignment for Selectel for the Frontend direction.

## Публикация на GitHub Pages

1. В репозитории: **Settings → Pages** ([ссылка](https://github.com/Julsssssssssssss/TestAssignmentSelectel/settings/pages)).
2. В блоке **Build and deployment** в поле **Source** выбери **GitHub Actions** (не «Deploy from a branch»).
3. Закоммить и запушь изменения в ветку `main` (в том числе файл `.github/workflows/deploy-pages.yml`).
4. После пуша открой вкладку **Actions** — запустится workflow «Deploy to GitHub Pages». Дождись зелёного статуса.
5. Сайт будет доступен по адресу: **https://julsssssssssssss.github.io/TestAssignmentSelectel/**
